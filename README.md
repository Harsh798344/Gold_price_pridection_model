# 🏆 Gold Price Prediction Model

A Machine Learning-based web application that predicts gold prices using historical market data and economic indicators. The project combines data analysis, feature engineering, model training, and web deployment to provide accurate gold price predictions.

## 📌 Project Overview

Gold prices are influenced by various economic and financial factors such as currency exchange rates, stock indices, crude oil prices, and market trends. This project uses Machine Learning algorithms to analyze these factors and predict future gold prices.

The application allows users to input market-related parameters and receive a predicted gold price through an interactive web interface.

---

## 🚀 Features

- Historical Gold Price Analysis
- Data Cleaning & Preprocessing
- Feature Engineering
- Machine Learning Model Training
- Gold Price Prediction
- Interactive Web Interface using Flask
- Real-Time User Input Prediction
- Model Performance Evaluation
